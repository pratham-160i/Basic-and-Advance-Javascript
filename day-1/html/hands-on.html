Hands-on: In-class Tasks

Hands-on Tasks

Complete these tasks during class. Open the browser console (F12) and try each exercise.

Task 1: Console Output

Log different types of information to the console:

console.log("Your name");
console.log(25);
console.log(true);
console.log(3.14);

Task 2: Simple Math

Perform basic arithmetic operations:

console.log(10 + 5);
console.log(20 - 8);
console.log(4 * 6);
console.log(15 / 3);
console.log(17 % 5);

Task 3: String Concatenation

Join strings together:

console.log("Hello " + "World");
console.log("I am " + 20 + " years old");
console.log("Price: " + 100 + " rupees");
